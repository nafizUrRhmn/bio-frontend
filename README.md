Agent Banking has built with Angular and latest Bootstrap 5. It is meant to provide the best possible Experience to User.
 
## Getting Started

1. Clone from Github 
```
git clone https://github.com/BBLRnD/frontend.git
```
2. Install packages
```
npm i --legacy-peer-deps
```

3. Run project
```
npm run start
```



## Technology Stack

 - Bootstrap 5
 - Angular 15
